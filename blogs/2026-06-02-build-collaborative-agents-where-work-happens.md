Microsoft announced new tools and capabilities for building AI agents in Teams, including the general availability of the Teams SDK in Python, JavaScript, and C#. Agents from partners like Linear, GitHub, and Cursor can now participate in conversations and complete tasks directly within Teams workflows, with expanded support for sovereign cloud environments like GCC High and DoD. New interaction patterns such as targeted messages and agent sessions enable agents to function as seamless teammates in everyday work.
